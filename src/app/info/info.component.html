  • About
    1. TranSphere is a web application that offers internal transfers on a half price. The application is formed by 2 modules: for users and for customers. Every user has its own profile page and has the ability to upload content on the platform. Every user is able to access this information for free.
    2. For Users: Every user in the app can upload offers for transfers and can edit them. The user recieves information about the status of his active offers.
    3. Registration: The registration form for users is located on the index page of the project. In order the registration to be successful the user should provide the needed information for authentication. After registration the user can access his profile via login page where he should type his email and driver license id.
    4. Adding an Offer: Every user on the platform is able to add offers. In order to add an offer tha user should click/tap on the red button with a white cross inside - this is a link to the content-upload page. The user should provide detailed information about the offer he wants to upload. After a successful upload of an offer the user can find a link to it on his profile page, where he can also edit it by clicking on the edit button next to the offer link.
    5. Editing an Offer: Every user on the platform is able to edit his offers. There is an edit page from where the user can change details about the offer. The only details that cannot be changed are: Arrive Time and Date of the transfer.
    6. Notifications panel: Every user has an individual notifications panel. It can be accessed from the main navigation of the platform. When a customer decides to book a transfer that is offered by user, the user recieves information about this request on the notifications panel. From there the user can confirm the booking and to send an automatic email to the customer.
  • For Customers
    1. For Customers:Every customer in the app can explore and/or book offers for transfers.
    2. Explore Transfers: Every customer is able to explore the offered transfers and to filter them by city or month. There is a page for a fast-search of transfers. When the customer clicks on an offer he is redirected to a page that contains detailed information about the transfer.
    3. Book Transfers: Every customer is able to book a transfer on the platform. The customer should go to the detail page related to the transfer and to scroll down to the "book" button. After clicking on this button the customer sends a request to the user. If the usr confirms the request the customer recieves a confirmation message. If not, the customer recieves another message.
